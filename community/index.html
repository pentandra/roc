Community

roc is open source, hosted on Github. If you want to help out, create a fork of the project and submit a pull request. Make sure to include tests with your contribution.

If you find a bug or have an issue, please file it with the issue tracker. The source code for this website is also available on Github. If you find a mistake or want to add a guide, submit a pull request or file an issue, whichever is most comfortable for you.

Don’t be shy! roc is a community project. We really want to change how research is done for the better—to design technology that fits the way research is done, making the process of doing and publishing research deeper, richer, and more meaningful. If you have a great idea or if we “just aren't getting it,” please help out and let us know!
